Software project kurssin harjoitustyö

Verkkokauppa
Projektissa on käytetty Java 17, Spring Boottia ja lisäksi ollaan käytetty lombokkia ja thymeleafia.
Käytöohjeet:
  - Ohjelma käynnistetään VerkkokauppaApplication.java tiedoston ajamalla.
  - Sivulle pääsee localhost/63184 sivulla.
  - Jos portti on käytössä, voit valita uuden portin application.properties kohdan server.port
  - Järjestelmän hallitsija voi kirjautua h2-consolen kautta kirjoittamalla localhost/63184/h2-console
  - Tuotteita voidaan lisätä varastoon h2-consolin avulla.
  - Tuotteita voidaan lisätä ostoskoriin ja niitä poistuu varastosta oston tapahtuessa.
